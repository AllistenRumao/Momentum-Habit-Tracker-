MOMENTUM HABIT TRACKER - PROJECT INFORMATION
================================================================================

PROJECT TYPE: React Application (Vite + React 18)

WHAT THIS IS:
A complete, production-ready habit tracking application that you can deploy
immediately to any hosting platform. All files are included and configured.

================================================================================
QUICK START
================================================================================

1. Install dependencies:
   cd momentum-habit-tracker
   npm install

2. Run locally:
   npm run dev
   (Opens at http://localhost:3000)

3. Build for production:
   npm run build
   (Creates 'dist' folder)

================================================================================

- Data persists between sessions
- Data is private to each browser/device
